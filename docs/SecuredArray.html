Class: SecuredArray

SecuredArray(…params)

Wrapper class is designed to secure arrays from unintended/easy mutation.
Security through complexity is applied.
Goal was to create an Array type that is not mutable from anywhere easy, but only from an app-namespace object(s) (the container object better) by methods designed for Array mutation purposes explicitly.
