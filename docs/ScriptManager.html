ScriptManager(urlsopt)

Methods

load(selector, url, positionopt)

Performs dynamic script(s) injection into the DOM of a document (file) it's being called at. Accepts one URL as well as many ordered URLs in Set.

Parameters:
Name Type Attributes Default Description
selector string

Valid CSS selector of a parent / sibling DOM element.
url string | Set.<string>

One or many URLs (arranged in the load order), relative to the document, that method is being called at.
position string <optional>
 "append"

Switch that specifies the position where script(s) will be injected. May be any of these values:

  • append - Default value. Script(s) will be appended to the childlist of the element, identified by selector.
    MDN: ParentNode.append();
  • prepend - Scripts will be prepended to the childlist of the element, identified by selector.
    MDN: ParentNode.append();
  • before - Scripts will be injected to the parent of the element, identified by selector, right before the element itself.
    MDN: ChildNode.before();
  • after - Scripts will be injected to the parent of the element, identified by selector, right after the element itself.
    MDN: ChildNode.after();
